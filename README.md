## How to Open Locally

브라우저에서 `index.html` 또는 `presentation.html`을 열면 됩니다.

프레젠테이션 조작:

- 좌우 방향키: 이전/다음 슬라이드
- Space: 다음 슬라이드
- F: 전체화면 전환
- 우측 하단 아이콘: 전체화면 전환
